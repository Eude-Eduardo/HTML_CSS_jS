Parágrafos e quebras de linha

Você pode escrever um parágrafos de qualquer jeito. basta colocar tudo no meio do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar expecífico
como esse, você vai usar a tag <br>

Vamos adicionar símbolos especiais:
Marca registrada: ®
Copyrigth: ©
Trade Mark: ™
Euro: €
Libra: £
Yen: ¥
Cent; ¢
Delta: δ
Seta esquerda tripla: ⇚
Seta esquerda: ←

vamos adicionar emoji: 😭 🚀
